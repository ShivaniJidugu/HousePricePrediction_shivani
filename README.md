# House Price Prediction

## Project Overview
This project focuses on predicting house prices using Machine Learning techniques. The dataset contains information about house features such as area, bedrooms, bathrooms, stories, parking spaces, furnishing status, and preferred location. The goal is to build regression models that can estimate house prices accurately.

## Dataset
- Dataset: Housing Prices Dataset
- Source: Kaggle
- File Used: Housing.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Tasks Performed
1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding
4. Model Training using:
   - Linear Regression
   - Random Forest Regressor
5. Model Evaluation using:
   - MAE
   - RMSE
   - R² Score
6. Data Visualization

## Results

### Linear Regression
- MAE: 970043
- RMSE: 1324507
- R² Score: 0.653

### Random Forest Regressor
- MAE: 1022560
- RMSE: 1401497
- R² Score: 0.611

Linear Regression performed better on this dataset.

## Visualizations
- Distribution of House Prices
- Correlation Heatmap
- Actual vs Predicted House Prices

## Conclusion
The project successfully predicted house prices using machine learning models. The analysis showed that area, bathrooms, stories, parking spaces, and preferred location have a significant impact on house prices. Linear Regression achieved the best overall performance and provided reasonably accurate predictions.
